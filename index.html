Branche ton clavier en USB puis clique sur Connecter un clavier MIDI. Les touches jouées s'allument en temps réel. Tu peux aussi cliquer les touches à l'écran pour les entendre.

Nécessite un navigateur compatible Web MIDI (Chrome, Edge, Opera sur macOS). Les instruments échantillonnés (piano, orgue…) sont chargés depuis gleitz/midi-js-soundfonts.
